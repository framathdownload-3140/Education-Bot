# 🎓 Education Bot

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/francismatheos/Education-Bot/blob/main/Streamlit_Chatbot_Edukasi_Frencis_Matheos_S.ipynb)

## 📖 Deskripsi
Education Bot adalah chatbot AI berbasis LLM untuk membantu pembelajaran dan edukasi.

##  Cara Menggunakan

### 1. Buka di Google Colab
Klik badge **"Open In Colab"** di atas untuk menjalankan langsung di Google Colab.

### 2. Langkah-langkah:
- Buka notebook di Colab
- Jalankan semua cell (Runtime → Run all)
- Dapatkan Groq API Key di https://console.groq.com/keys
- Masukkan API Key Anda
- Akses chatbot melalui URL ngrok yang muncul

## 📋 Fitur
- ✅ Chatbot edukasi interaktif
- ✅ Multi-model AI (Llama 3.3, Mixtral, dll)
- ✅ UI modern dan responsif
- ✅ Akses via ngrok tunnel

## 🛠️ Teknologi
- Python
- Groq API
- Flask
- ngrok
- Google Colab

## 📝 Requirements
- Groq API Key (gratis)
- Akun Google untuk Colab
- Ngrok account (opsional)
